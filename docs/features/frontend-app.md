# 前端应用功能文档

## 概述

Streamlit 交互式前端，提供数据同步、回测运行和结果展示功能。

## 功能说明

### 1. 侧边栏控制区

- **🔄 同步最新数据**: 点击按钮调用数据管理器同步行情数据
- **📅 回测日期**: 选择回测的开始和结束日期
- **💰 初始资金**: 设置回测的初始资金
- **🚀 运行回测**: 执行回测逻辑

### 2. 主界面

#### CR5% 走势图

- 使用 Plotly 绘制历史 CR5% 走势
- 显示 40% 的风控阈值线

#### 回测结果展示

- **核心指标卡片**: 总收益率、年化收益率、最大回撤、夏普比率
- **资金曲线**: Plotly 绘制权益走势
- **交易日志**: 展示详细的交易记录

## 运行方式

```bash
streamlit run app/app.py
```

## 技术栈

- Streamlit 框架
- Plotly 可视化库
- Pandas 数据处理

## 界面特点

- 响应式布局
- 直观的操作体验
- 实时状态反馈
